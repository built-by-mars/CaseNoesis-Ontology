# CQ-13 through CQ-26 — Cross-language graph API parity

**Date:** 2026-07-12  
**Scope:** `python/case_uco/graph.py`, `csharp/CaseUco/CaseGraph.cs`, `java/.../CaseGraph.java`, `rust/src/graph.rs` (+ tests)  
**Do not tag / do not push** (per directive).

## Summary

Composition APIs now share deep-copy read surfaces, named duplicate policies, one property-conflict engine, transactional load, safe relationship IDs, multi-type triple counts, UTF-8 Java I/O, and typed deserialization diagnostics. Rust clippy `unwrap_used` / `expect_used` issues in `graph.rs` are cleared.

## CQ disposition

| CQ | Status | Implementation |
|----|--------|----------------|
| **CQ-13** | Done | `get()` returns a **deep** copy in Python/C#/Java (nested lists/maps not shared). Nested-mutation tests added (Python + C#). Rust `get` returns owned `Map` clone. |
| **CQ-14** | Done | `upsert_node` / `create_relationship` return deep copies / owned `Value` clones — not live internal maps. |
| **CQ-15** | Done | Named policies in all four languages: `reject`, `merge_identical`, `merge_compatible`, `replace` (`error` → `reject`). Boolean `RejectDuplicates` retained as a thin compatibility shim. |
| **CQ-16** | Done | Shared conflict engine (`_apply_property` / `ApplyProperty` / `applyProperty` / `apply_property`) used by load, upsert, and add-property. |
| **CQ-17** | Done | Explicit `set_property` / `SetPropertyValue` / `set_property` (replace mode). `merge_compatible` raises on conflicting scalars (no silent array coercion). |
| **CQ-18** | Done | Semantic equality for `@id` refs, typed literals (`xsd:` vs full XSD IRI), and set-like `@id` arrays before conflict decisions. |
| **CQ-19** | Done | Transactional `load`: snapshot context+nodes+index; restore on any failure. |
| **CQ-20** | Done | Constructor `extra_context` / `AddContext` / load merge all reject prefix collisions (including vs standard defaults). |
| **CQ-21** | Done | `split` rejects ≤0 with typed errors (`InvalidSplitSizeError` / `ArgumentOutOfRangeException` / `IllegalArgumentException` / `GraphError::InvalidSplitSize`). Rust does not panic. |
| **CQ-22** | Done | Relationship kind slug: constrained charset `[A-Za-z0-9._-]`, length cap 64. |
| **CQ-23** | Done | Optional `relationship_id` / `assertion_id` so identical source/target/kind can coexist. |
| **CQ-24** | Done | `estimate_triples` counts every `rdf:type` in multi-type arrays; refs/literals defined vs embedded nodes. |
| **CQ-25** | Done | Java file/process I/O uses explicit UTF-8 (`OutputStreamWriter`/`Files.readString`/`StandardCharsets.UTF_8`); no `FileWriter` / platform-default `new String(bytes)`. |
| **CQ-26** | Done | Typed `DeserializationWarning` diagnostics when rehydration falls back to raw (Python/C#/Java). |
